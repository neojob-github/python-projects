1. Игра не для продажи
2. Цель игры - убить босса который появляется при сложности = 25.0, либо набрать максимальное количество очков для зала славы
3. Для нанесения урона по врагам могут использоваться: Пулемёт, Дробовик, Лазер (Выбрать пушку на 1, 2, 3, либо колесиком мыши)
4. Оружие можно прокачивать на "SPACE", в случае, если хватает очков (стоимость прокачки = UP или очки станут желтыми)
5. В игре можно передвигать персонажа с помощью "WASD"
6. Для выстрела используется ЛКМ (можно зажимать ЛКМ для непрерывной стрельбы)
7. Креветка лечит игрока
8. Персонаж может получить урон от зомби. Если здоровье опустится ниже 0 — игрок проиграл. Рекорды обновятся.
9. В рекорды записывается количество очков игрока на момент убийства босса, либо на момент смерти игрока.
10. Открыть игру с помощью python IDLE и подобного ПО

Игра сделана на базе библиотеки python3 pygame

Для запуска игры нужны библиотеки:
pygame, sys, random, os, numpy
getpass, json, pandas
math
